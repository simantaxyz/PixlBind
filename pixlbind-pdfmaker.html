Privacy Policy
LAST UPDATED — JANUARY 2026
✦ ON-DEVICE ONLY

PixlBind is built on a simple principle: your files never leave your device. All PDF generation happens entirely in your browser.

1. No Data Collection

PixlBind PDF Maker does not collect, store, transmit, or process any personal data or file data on any server. All PDF generation runs entirely in your browser using standard Web APIs (Canvas API, FileReader, Blob). Your images are processed in local browser memory and immediately discarded once you close or refresh the page.

2. No Cookies or Tracking

We do not use tracking cookies, analytics cookies, or any other persistent identifiers. We do not embed any third-party analytics tools.

3. Third-Party Services

The only external resource loaded is the Google Fonts stylesheet. No file content or personal data is involved in this request. The jsPDF library is loaded from CDN to enable PDF generation — it operates entirely client-side.

4. File Security

  • Images are read into browser memory via the FileReader API
  • PDF generation is performed using the jsPDF library in your browser
  • Output is generated as a Blob URL in your browser
  • All data is cleared from memory when you navigate away
  • No file ever touches our infrastructure — because we have none
